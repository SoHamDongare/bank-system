# Bank System (Python OOP Project)

## 🎯 Goal

To understand how real backend systems manage users, data, and operations using Python.

## 📌 Description

This is a simple Bank System built using Python and Object-Oriented Programming (OOP).
It simulates basic banking operations such as account creation, deposits, withdrawals, and balance management.
This project was built as part of my backend development learning journey.

## 🚀 Features

* Account creation and management
* Deposit and withdrawal operations
* Balance tracking
* Input validation and error handling

## 🛠️ Technologies Used

* Python
* OOP (Classes & Objects)

## 📂 Project Structure

* `bank_system.py` → Main program file

## ▶️ How to Run

1. Open terminal
2. Run:

```
python bank_system.py
```

## 📚 What I Learned

* Designed a system using Object-Oriented Programming (OOP)
* Implemented real-world business logic (banking operations)
* Applied input validation and exception handling
* Structured code for clarity and scalability 

## 💡 Future Improvements
